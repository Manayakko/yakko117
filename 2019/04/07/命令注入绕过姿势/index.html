    使用 < 或 >

    cmd <> file 以读写模式将文件 file 重新定向到输出，文件 file 不会被破坏。只有当应用程序利用这一特性时，它才有意义。

    $IFS

    IFS 是一种set变量， 当shell处理‘命令替换’和‘参数替换’，shell根据IFS的值，默认是 space， tab， newline 来拆解读入的变量， 然后对特殊字符进行处理， 最后重新组合赋值给变量。

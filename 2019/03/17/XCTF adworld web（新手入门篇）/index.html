simple_php

题目描述：小宁听说php是最好的语言,于是她简单学习之后写了几行php代码。

php

前半段可以用：

http://111.198.29.45:30668/index.php?a=a

同理 var_dump('1234a' == 1234)
结果为 true ,php的弱类型比较会忽略字母a

所以后半段可以用：

http://111.198.29.45:30668/index.php?a=a&b=1235a

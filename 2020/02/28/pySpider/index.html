爬虫学习

思路

  • 向浏览器发送请求，获取响应
  • 得到 200回显 后，需要进行html解析
    • 创建一个新的文件来解析 html(通过正则匹配
    • 调用 bs4 解析我们的html （html5lib –可以通过浏览器的形式取解析网页）
    • 需要去获取我们需要的信息，例如：图片 -> img 标签 -> src属性值
    • 再通过src路径向浏览器发送请求
    • 请求成功以后,将我们的图片信息以二进制的方式进行写入我们的文件中
    • 实现批量爬取

如何得到 flag.php 文件名

通过拜读飘零师傅的文章，知道了可以利用 getallheaders() 拿到环境变量
使自己传参的参数变为可控的，进而执行命令。

也可以利用常规方法，从找个函数来返回 flag.php 文件名
scandir() 函数可以扫描当前目录下的文件，例如：

php

按照飘零师傅的思路，利用 getcwd() 获得当前目录，scandir(getcwd()) 获得当前目录的文件。
而如果要往上层目录看的话，就利用dirname()，使用scandir(dirname(getcwd()))。
但是这里的 et 被过滤，所以我们要找其他的方法

因此 scandir 的 . 如何利用成了问题

localeconv() 函数返回一包含本地数字及货币格式信息的数组

如何得到flag.php

如上，先利用 current(localeconv()) 得到当前目录

Code

可以看到 flag.php 是倒数第二个值，假设是倒数第一个我们可以用end()，但是并没有一个操作数组的函数能够输出数组的倒数第二个值。如何解决，看了大佬的文章总结些方法

array_reverse()

以相反的元素顺序返回数组

Code

然后通过操作数组的方法，next() 将内部指针指向数组中的下一个元素并输出
next(array_reverse(scandir(pos(localeconv())))) 就得到了 flag.php

array_rand(array_flip())

array_flip() 交换数组的键和值

Code

array_rand() 从数组中随机取出一个或多个单元，不断刷新访问就会不断随机返回，本题目中scandir()返回的数组只有5个元素，刷新几次就能刷出来 flag.php

Code

session_id(session_start())

通过飘零师傅的文章可以知道，题目虽然ban了 hex 关键字，导致 hex2bin() 被禁用，但是我们可以并不依赖于十六进制转ASCII的方式，因为 flag.php 这些字符是 PHPSESSID 本身就支持的。

使用 session之前需要通过 session_start() 告诉PHP使用session，php默认是不主动使用session的。

session_id() 可以获取到当前的session id。

因此我们手动设置名为 PHPSESSID 的cookie，并设置值为 flag.php
